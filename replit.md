# AI Strategy OS

## Overview

AI Strategy OS is a static website — the first implementation layer of Genesis Protocol. It presents the project's purpose, principles, and its first module (AI Sports Lab), with documentation kept in Markdown files.

## Project Structure

- `index.html` — landing page (links to docs and modules)
- `style.css` — landing page styles (dark theme)
- `README.md` — project overview
- `docs/` — roadmap and genesis principles (Markdown)
- `sports-lab/` — AI Sports Lab module docs (README, model plan)
- `data/` — placeholder for future datasets

## Running

Served as a static site via the "Static Site" workflow (`python3 -m http.server 5000`).
