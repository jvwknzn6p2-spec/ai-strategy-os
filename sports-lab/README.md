# AI Sports Lab

AI Sports Lab is the first experimental module of AI Strategy OS.

## Focus

- MLB game prediction
- Monte Carlo simulation
- Data collection
- Odds analysis
- Expected value calculation
- Backtesting
- AI multi-agent review

## Goal

Build a transparent, testable prediction pipeline where every output
can be measured against real outcomes.
